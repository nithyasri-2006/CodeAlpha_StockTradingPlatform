# Stock Trading Platform

This is a Java console-based Stock Trading Platform developed as part of the **CodeAlpha Java Internship (Task 2)**.

The application simulates a basic stock trading environment where users can view market stocks, buy and sell stocks, and track their portfolio balance using a menu-driven interface.

---

## Features

- View available market stocks with prices  
- Buy stocks from the market  
- Sell owned stocks  
- View portfolio details and current balance  
- Menu-driven console interface  
- Uses Object-Oriented Programming concepts  

---

## Technologies Used

- Java  
- VS Code  
- Object-Oriented Programming (OOP)  
- ArrayList  
- Scanner (User Input)
